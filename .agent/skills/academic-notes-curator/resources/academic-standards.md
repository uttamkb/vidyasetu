# Academic Standards & Syllabus Boundaries

## General Standards (The Premium Quality)
1. **Premium Pedagogy:** Content must be extremely engaging, using real-world analogies, step-by-step logical breakdowns, and intuitive mathematical derivations (similar to top-tier coaching modules).
2. **Conceptual Clarity:** Prioritize deep understanding over rote memorization. If a concept is abstract, visualize it with textual descriptions or structured formatting.
3. **Objective Tone:** Maintain an authoritative yet accessible "master teacher" tone.

## Strict Syllabus Boundaries (The NCERT Shield)
1. **Board-Focused:** The content MUST strictly adhere to the official NCERT / CBSE syllabus for the target grade. 
2. **NO Out-of-Syllabus Bloat:** Premium coaching materials (like Allen, Byju's, etc.) frequently bleed into JEE/NEET competitive domains. **You must actively filter this out.** If a concept, formula, or historical tangent is NOT in the official NCERT textbook, do NOT include it in the core notes.
3. **Scope Management:** If a concept is complex but required, teach it beautifully. If it is complex and *not* required by NCERT, omit it entirely to prevent student overwhelm.

## Citation & Formatting
1. **Terminology:** Always bold and explicitly define key NCERT vocabulary.
2. **Formatting:** Use markdown extensively (tables, bullet points, bolding) to make the text scannable and easy to digest.
