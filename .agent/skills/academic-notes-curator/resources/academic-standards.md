# Academic Standards & Citation Rules

## General Standards
1. Use objective, academic tone.
2. Ensure summaries are devoid of personal opinion.
3. Highlight definitions of key terms explicitly in the Terminology section.

## Citation Rules
1. **Primary Sources:** Cite all primary sources correctly using APA format unless specified otherwise.
2. **In-text Citations:** Include in-text citations for direct quotes and specific data points (Author, Year, p. X).
3. **References:** Provide a complete bibliography at the end of the document if multiple sources are synthesized.
